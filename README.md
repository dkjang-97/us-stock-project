# 📈 AI 기반 미국 주식 재무 및 동향 분석 시스템 (US Stock Dashboard)

Streamlit과 FastAPI 백엔드를 결합한 비동기 폴링 구조의 미국 상장 기업 재무제표 및 뉴스 동향 분석 대시보드 시스템입니다. 
Gemini AI를 활용한 실시간 기업 프로필 번역, 뉴스 순차 번역/요약 및 투자 편향성 자동 검증 필터링 파이프라인이 탑재되어 있습니다.

---

## 🌟 주요 특징 및 아키텍처

### 1. 고해상도 분기 재무 정보 추이 ($B 포맷)
- **SEC EDGAR API** 연동을 통해 3개년(최근 12분기)의 상세 10-Q 및 10-K 공시를 자동 파싱합니다.
- 복잡하고 긴 숫자를 가독성 높은 **십억 달러 ($B)** 단위로 통일하여 KPI 스코어카드 및 인터랙티브 Plotly 차트, 실적 요약 테이블에 반영합니다.

### 2. 호재(Good) vs 악재(Bad) 2단 뉴스 분석 레이아웃
- 복잡함을 야기하는 중립 뉴스를 과감히 제외하고, 기업 실적과 미래 가치에 영향을 주는 **호재(좌측)와 악재(우측) 1:1 대비 컬럼**으로 뉴스 섹션을 배치했습니다.
- 3개년(2023, 2024, 2025)의 연도별 탭 필터 분류 및 **전체 연도 보기** 기능이 제공되어 스크롤로 역대 중요 기사 추이를 파악할 수 있습니다.
- 차트 위에 주요 뉴스 발생 시점을 **세로 점선**과 느낌표 아이콘으로 연동하여 마우스 오버 시 한글 기사 요약 툴팁이 팝업됩니다.

### 3. 고품질 순차 번역(Sequential) 및 API Rate Limit 방지
- 번역 누락을 막고 정밀성을 보강하기 위해 개별 기사 단위로 순차적인 AI 번역(`gemini-1.5-flash`)을 실행합니다.
- API 요청 부하 및 차단을 안정적으로 회피하기 위해, 기사당 번역 요청마다 **1.2초의 Cooldown Timeout (`time.sleep(1.2)`)**을 의무 적용했습니다.

### 4. 이중 AI 실시간 검증/보정 파이프라인
- 리포트 및 기사 내 극단적 추천 문구(`강력 매수`, `무조건 폭등` 등)의 편향성을 **이중(Local 정규식 필터 -> Gemini 검증)**으로 체크합니다.
- 편향 발견 시 로컬 교정 규칙 및 API 재작성을 시도하여 최종 대시보드 리포트에는 철저하게 사실에 기반한 정돈된 텍스트만 노출되도록 보장합니다.

## 🛠️ 설치 및 시작 방법

### 1. 가상환경 설정 및 의존성 패키지 설치
```bash
# 가상환경 활성화 (MacOS)
source venv/bin/activate

# 의존성 패키지 설치
pip install -r requirements.txt
```

### 2. 환경 변수 설정
루트 디렉토리에 `.env` 파일을 생성하고 아래와 같이 Gemini API Key를 지정해 줍니다:
```env
GEMINI_API_KEY="YOUR_GEMINI_API_KEY_HERE"
```

### 3. 백엔드 서버 구동 (FastAPI - Port 8000)
```bash
uvicorn main:app --host 127.0.0.1 --port 8000
```

### 4. 프론트엔드 대시보드 구동 (Streamlit - Port 8501)
```bash
streamlit run app.py --server.port 8501
```

## 📂 프로젝트 구조
- `app.py`: Streamlit 대시보드 엔트리포인트 (로딩 상태 비동기 처리)
- `visualizer.py`: Plotly 차트, 연도별 탭, 호/악재 2열 카드 및 표 레이아웃 구현
- `main.py`: FastAPI 엔드포인트 설계
- `tasks.py`: 백엔드 비동기 백그라운드 태스크 및 이중 검증 파이프라인 제어
- `explainer_trend.py`: 뉴스 수집, 순차 AI 번역기 및 역사적 핵심 이벤트 DB 구축
- `data_load.py`: SEC API 호출 및 분기 재무 정보 가공 엔진
- `ai_verify.py`: 로컬 정규식 편향 탐지 및 Gemini AI 리포트 재합성 모듈
- `test_integration.py`: 시뮬레이션 기반 종단 간 통합 테스트 스크립트
